# Daily Roll

A 3D block-rolling puzzle: tip a 2×1 block end-over-end across a floating tile
island and land it standing upright in the glowing hole — the level is carved
deterministically from the date, so everyone gets the same puzzle (and the same
par) each day.

Built with **three.js** (WebGL). No sound.

## How to play

- **Swipe** on the board, **tap the on-screen arrows**, or use **arrow / WASD**
  keys to roll the block one tile at a time.
- The block must finish **standing upright on the teal hole** to win.
- Roll off an edge and it tumbles into the void and resets to the start.
- **Par** is the fewest possible moves (computed by an exhaustive search). Undo
  and Reset are always available; your best result for the day is saved locally.

## How it works

The block's grid state (standing / lying along X / lying along Z) and its
rolling transitions are modelled exactly; the day's floor is generated by a
seeded reverse random walk from the goal, which guarantees the puzzle is
solvable, and a breadth-first search finds the optimal move count for par. The
date comes only from the app's folder-name slug in the URL — no system clock,
no hardcoded date.

## Run locally

```bash
npm install
npm run dev      # development
npm run build    # type-check + production build into dist/
npm run preview  # serve the production build
```
